Outdoorsy

Designed to make coordinating outdoor recreation easy. Using React and the Google Places API, we pulled the top hiking, biking, and outdoor activitiesto save and schedule upcoming destinations with friends. Version 2.0 will send an email directly to invitees. Version 2.0 will allow users to “follow” each other’s Outdoorsy activity

Technology

React | Javascript | HTML | CSS | MongoDB | Google Places API
